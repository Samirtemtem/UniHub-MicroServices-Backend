Documentation des Services UniHub

La plateforme UniHub est composée de plusieurs microservices qui travaillent ensemble pour fournir une expérience complète aux utilisateurs. Cette documentation est organisée en deux sections principales pour répondre aux besoins de différents publics.

  • Documentation Générale - Pour les utilisateurs et administrateurs de la plateforme qui ont besoin de comprendre les fonctionnalités et l'utilisation générale.
  • Documentation Technique - Pour les développeurs qui ont besoin de détails précis sur les API, formats de données et intégrations.
